## Docker構成

### docker-compose.yml (MVP)

```yaml
version: '3.8'

services:
  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile
    ports:
      - "8000:8000"
    environment:
      - LM_STUDIO_URL=[http://host.docker.internal:1234/v1](http://host.docker.internal:1234/v1)
      - LM_STUDIO_MODEL=qwen2.5-coder-7b-instruct
    volumes:
      - ./backend:/app
    command: uvicorn main:app --host 0.0.0.0 --port 8000 --reload
    networks:
      - agentic-network

  frontend:
    build:
      context: ./frontend
      dockerfile: Dockerfile
    ports:
      - "8501:8501"
    environment:
      - BACKEND_URL=[http://backend:8000](http://backend:8000)
    volumes:
      - ./frontend:/app
    depends_on:
      - backend
    command: streamlit run [app.py](http://app.py) --server.port 8501
    networks:
      - agentic-network

networks:
  agentic-network:
    driver: bridge
```

### backend/Dockerfile

```docker
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8000

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

### frontend/Dockerfile

```docker
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

EXPOSE 8501

CMD ["streamlit", "run", "[app.py](http://app.py)", "--server.port", "8501"]
```

## LM Studio設定

### インストール

1. [LM Studio](https://lmstudio.ai/) からダウンロード
2. アプリケーションを起動

### モデルダウンロード

1. "Discover" タブを開く
2. "qwen2.5-coder-7b-instruct" を検索
3. ダウンロード

### サーバー起動

1. "Local Server" タブを開く
2. モデルを選択
3. "Start Server" クリック
4. ポート 1234 で起動確認

### OpenAI互換モード

- デフォルトで有効
- エンドポイント: [`http://localhost:1234/v1`](http://localhost:1234/v1)

## AWS設定 (Phase 2)

### AWS CLIインストール

```bash
pip install awscli
aws configure
```

### 設定項目

- AWS Access Key ID
- AWS Secret Access Key
- Default region: ap-northeast-1
- Default output format: json

### 必要なIAM権限

- Bedrock: `bedrock:InvokeModel`
- S3: `s3:PutObject`, `s3:GetObject`
- RDS: 接続権限
- Lambda: `lambda:InvokeFunction`
